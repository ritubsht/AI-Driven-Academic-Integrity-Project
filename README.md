# AI-Driven-Academic-Integrity-Project
# This project explores academic integrity in the age of AI using data analysis and the PAIEL framework using Machine Learning and dashboard integration. It analyzes student awareness and usage of AI tools through survey data and performs exploratory analysis on AI-generated vs human-written text datasets to understand patterns relevant to academic integrity.

## PAIEL Framework
# The PAIEL framework (Policy, Awareness, Intervention, Ethics, and Learning) to promote responsible AI usage in education. It aims to ensure that the integration of AI tools in academic environments supports learning while maintaining academic integrity.

### Notebooks
# AI Awareness Survey Analysis – Exploratory analysis of Google Forms data to study students’ awareness and usage of AI tools.
# AI vs Human Text Dataset Analysis – Exploratory analysis of a Kaggle dataset containing AI-generated and human-written text.

#### Tools
# Python, Pandas, NumPy, Matplotlib, Seaborn etc..

##### Future Work
# Train and optimize machine learning models for AI vs Human text classification and integrate results into an analytical dashboard using PAIEL framework to support academic integrity monitoring.
